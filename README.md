# PRIMA_update

## Установка зависимостей

Установите необходимые пакеты с помощью `pip`:

```bash
pip install -r requirements.txt
```

Среди зависимостей теперь есть библиотека `rich`, обеспечивающая:
- цветной и структурированный вывод сообщений в консоли;
- интерактивные запросы (`Prompt`, `Confirm`);
- таблицы, панели и прогресс-бары для отображения статуса операций.

## Новые возможности интерфейса

Интерфейс скрипта обновления PRIMA использует `rich` для:
- визуального отображения изменений в виде таблиц;
- интерактивного выбора действий через панель меню;
- наглядного прогресса копирования и синхронизации файлов.

Если вы запускаете приложение в терминале без поддержки ANSI-цветов, рекомендуется использовать современные консоли (Windows Terminal, PowerShell 7+ или аналоги), чтобы Rich корректно отображал стили.
