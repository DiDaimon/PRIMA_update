# План по улучшению удобства установки программы PRIMA Updater
Переход с install.bat файла на python скрипт и интеграцию с проектом:
 - создать модуль install.py.
 - в prima_update.py добавить проверку зависимостей (try except)
 - запускать установку зависимостей при провале проверки.
 - в случае неудачи загрузки зависимостей через pip(потому что не настроен прокси), спросить у пользователя ввод адреса прокси, логин, пароль.
 - использовать библиотеку rich.
